# Symbols

**♪X** = phonetic element X

**X (\> Y)** = form X distorted/changed into the form Y

**{X}** = word which is usually written as X in the modern orthography
(first in the entry is the word for which the glyph was originally
created, even if unattested)

**{X} ♪→ {Y}** = initially created for the word X, then phonetically
borrowed to write the word Y

**?** = uncertain

**\[X\]** (on pictures) = either some separate glyph X, or some compound
glyph X from which an element was taken for a picture

**X\*** = character usually transcribed as X, but its true nature is uncertain
